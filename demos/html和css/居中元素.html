水平居中

  • 行内元素: 在父容器设置text-align:center
    • 行内子元素
  • 块级元素

    • 方法1: 在子容器设置margin:0 auto，同时确保该子容器固定宽度（只居中子容器，不居中子元素）

    块级子元素

    方法2: 在子容器设置margin: 0 auto; display:table(table类似block元素但宽度为内容宽)

    块级子元素

    方法3: 先将父元素设置为相对定位，再将子元素设置为绝对定位，向右移动子元素，移动距离为父容器的一半，最后通过向左移动子元素的一半宽度以达到水平居中。transform属于css3内容，兼容性存在一定问题，高版本浏览器需要添加一些前缀

    块级子元素

    方法4: 通过CSS3中的布局利器flex中的justify-content属性来达到水平居中

    块级子元素

    方法5: 父元素flex布局，子元素居中

    块级子元素
  • 多块级元素 在父容器设置display: flex;justify-content: center;等同于子元素设置display:inlin-block,父容器设置text-align:center
